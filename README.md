# Grocery Sales Demand Forecasting and Anomaly Detection

This project analyzes grocery store sales data to understand demand trends and identify unusual sales days using machine learning.

## What this project does
- Aggregates transaction-level sales data into daily sales
- Learns demand patterns using a machine learning model
- Detects unusual sales days using anomaly detection
- Visualizes results using a Jupyter Notebook and a Streamlit dashboard

## Tools used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Streamlit
